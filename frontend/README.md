Se instalo:

npm install react-router-dom

    react-router-dom es la biblioteca estándar para implementar enrutamiento dinámico en aplicaciones web React, permitiendo la navegación entre diferentes componentes o páginas sin recargar el navegador

npm init -y
npm install express mongoose cors dotenv
npm install -D nodemon

    express: El framework para crear las rutas.

    mongoose: La librería para conectar con MongoDB.

    cors: Vital para que tu frontend de React pueda hacer peticiones al backend sin errores de seguridad.

    dotenv: Para guardar contraseñas y configurar el puerto de forma segura.

    nodemon: Para que el servidor se reinicie solo cada vez que guardes un cambio.\

npm install axios
    
    Aunque podrías usar fetch (que viene nativo en el navegador), Axios es mucho más cómodo para manejar errores y es lo que más se usa en la industria.


npm install -D tailwindcss@3 postcss autoprefixer
    Instalamos tailwind v3

npx tailwindcss init -p
    Inicializamos el tailwind v3
